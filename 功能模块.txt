核心功能
1. 用户认证与权限管理
•	使用 JWT 实现无状态认证。
•	用户登录后，Token 存储于客户端 LocalStorage，并在每次请求时进行携带。
•	后端验证 Token 并通过用户角色控制访问权限。
2. 设备管理模块
•	功能：设备信息管理，包括设备名称、类型、状态、位置等。
•	支持设备 增、删、改、查。
•	提供设备状态的快速查询和筛选功能。
•	使用 MySQL 存储设备信息，配合 Redis 做设备列表缓存，提升查询速度。
3. 工单管理模块
•	功能：支持工单的创建、分配、进度跟踪、状态更新等。
•	工单状态包括“待处理”、“处理中”、“已完成”。
•	采用 乐观锁 防止并发更新带来的数据冲突。
•	使用 MySQL 存储工单信息，使用 RabbitMQ 异步处理工单的相关通知。
4. 数据统计与报表
•	功能：统计设备的使用情况、工单的完成情况，生成报表。
•	按日期和设备类型统计工单数量、工单完成率等关键数据。
•	使用 MySQL 的聚合函数做统计，使用 Redis 缓存统计结果，提高查询效率。
5. AI 辅助诊断模块
•	功能：根据设备异常状态，调用 AI 大模型接口，提供设备故障的可能原因和维修建议。
•	使用 HttpClient 请求 AI 服务，处理返回的诊断结果。
•	结果缓存至 Redis，防止频繁调用 AI 服务。
